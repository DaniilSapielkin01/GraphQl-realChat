# Project Setup

To run the project, follow these steps:

## Server
1. Go to the `server` folder:
   ```bash
   cd server

2. Install dependencies:
yarn install
# or
npm install


3. Start the server:

yarn start
# or
npm start

The server runs on port 4000.

## Client

1. Navigate to the client folder.

yarn install
# or
npm install

2. Start the client:

yarn start
# or
npm start

Runs on port 8080.

## Home Page (Admin View)

1. Navigate to the home-page folder.

yarn install
# or
npm install

2. Start the admin view:

yarn start
# or
npm start

Runs on port 8081.
